How to install a PHP-based web application.

First, create an Amazom Lighsail Account.

Then sign into Lightsail.

Then go to the "Create Instance" area and create an instance.

Now choose Linux/Unix as the platform, choose Apps + OS as the blueprint, and then Wordpress as the template for this example.



Now choose your instance plan. You can get the first three months free with the $3.50, $5 or the $10 plan.



Your instance will take a few minutes to be created. After it's created you need to access Bitnami to continue setting up your web server.

In order to access the Bitnami, you need the special password to get in. Make sure you remember it so you can sign in on Word Press.
